  gitTutorial:
  1. Download software: https://desktop.github.com/
  2. Install software
  3. Create new Github Repository https://github.com/new or select existing Repository
  4. Copy Repository URL by pressing "Clone or download", for examle: https://github.com/Best-ITPro/gitTutorial.git
  5. Create folder for GitProjects in your PC or Mac, for example: D:\!My_Progs!\!GitHub!
  6. Run CMD (in Windows) or Terminal (im Mac or Linux) and go to folder: cd D:\!My_Progs!\!GitHub!
  7. By CMD use command: git clone https://github.com/Best-ITPro/gitTutorial.git for cloning repository in your comp
  8. Go to new created folder cd gitTutorial (D:\!My_Progs!\!GitHub!\D:\!My_Progs!\!GitHub!)
  9. Use git status to show status of this repository
  After this you can modify files in gitTutorial or create new files. Commands for work:
  1. git add *.* (or file name) - add file contents to the index
  2. git commit -m "Add file: filename" - record changes to the repository
  3. git push - update remote refs along with associated objects
  Work in another comp:
  1. Go to project folder in your comp or creat new
  2. By CMD use command: git clone https://github.com/Best-ITPro/gitTutorial.git for cloning repository in your comp
  3. git pull - fetch from and integrate with another repository or a local branch
  Enjoy it! ;)
